### 含义

```
public class Main {
    public  static void main(String[] agrs){
        //面向对象
        /*
           类的构造器
            1如果没有显示的定义类的够再起，则默认空的构造器
            2定义够再起格式：权限修饰符 类名（参数列表）{}
            3一个类中多个构造器，彼此构成重载
            4 一旦我们显示的定义了累的构造器，系统就不在提供默认的空构造器
            5 一个类中至少有一个构造器



            总结：属性赋值的先后顺序
                1.默认初始化
                2.显示初始化
                3.构造器赋值
                4 通过对象。方法 或者 对象。属性 方式 赋值



         */


    }

    public Main(){
        System.out.println("我是构造器");
    }

}
```
